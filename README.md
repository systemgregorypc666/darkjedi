🌌 Dark Jedi Search Engine
Desarrollador Principal: José Gregorio Hernández Calderón
Organización: System Gregory PC
Propósito: Motor de búsqueda independiente y especializado en biotecnología, fusión y neuroplasticidad.
📝 Descripción
Dark Jedi es un motor de búsqueda de alta precisión diseñado para operar de forma independiente. Su objetivo es indexar y filtrar información científica crítica, garantizando la privacidad del investigador y la integridad de los datos mediante el escudo de Cloudflare.

🛠️ Stack Tecnológico
Lenguaje de Rastreo: Python (Crawler/Spider).

Indexación: Elasticsearch para búsquedas en milisegundos.

Entorno de Desarrollo: Visual Studio Code.

Infraestructura: Dominio asegurado bajo genesis-ai.org.

🚀 Instalación y Uso
Clonar el repositorio en tu Chromebook.

Configurar las credenciales de la API de Cloudflare para protección de red.

Ejecutar el rastreador para iniciar la indexación de documentos de biotecnología.
